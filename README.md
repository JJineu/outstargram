## Overview
Nextjs13 SSR 을 활용하여, 인스타그램을 참고하여 만든 사이트입니다. 
<br><br>

## 개발 환경
- nextjs 13.3.0
- react-dom 18.0.11
- socket.io 4.7.0
- socket.io 4.7.0
- tailwindcss 3.3.1
- swr 2.1.3
